No hub de Java, cada arquivo de código ficará dentro de uma subpasta do seu respectivo tema.
Isso porque para cada arquivo de código que vamos praticar, será criado um src, que vem do IntelliJ...
Então ao invés de: 
Java HUB > 01. conceitos básicos > HelloWorld.java 

A organização será:
Java HUB > 01. conceitos básicos > 01. hello world (nome do conceito praticado) > src > HelloWorld.java

Assim é bom porque, apesar de criar mais um nível de diretório, vai deixar os arquivos de código mais organizados e fácil para consultar futuramente.
