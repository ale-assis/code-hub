# 📚 Code Hub: Minha Biblioteca de Snippets

Repositório pessoal de **snippets de código** coletados ao longo da minha jornada como desenvolvedor.  
Cada trecho vem de livros, cursos, vídeo-aulas e projetos reais, organizado por linguagem e tópico para servir de **referência rápida** sempre que eu precisar.

---

## 🎯 Objetivo

Construir uma **biblioteca vasta e organizada** de código que funcione como material de consulta pessoal.  
A ideia é simples: toda vez que eu aprender algo novo — seja um pattern, uma função útil ou uma solução criativa, o snippet vai parar aqui.

### Como o Code Hub ajuda?
- 🔍 **Consulta rápida**: encontrar exemplos reais sem precisar pesquisar do zero  
- 🧠 **Reforço de aprendizado**: revisar código já escrito consolida o conhecimento  
- 📦 **Backup centralizado**: tudo em um só lugar, versionado com Git  

---

## 🚀 Como usar

1. Navegue pela pasta da linguagem/tecnologia desejada 
2. Encontre o tópico específico dentro da subpasta
3. Consulte o snippet e adapte para o seu contexto
